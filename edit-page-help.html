Use Version Conrol

GIT

For any type of programming, it is important to use version control. Git is an open source Version Control system that is used by a number of companies and students. Version Control allows us work on code in teams, and manage changes. Without if, it would be dificult to manage all of the different versions of files, and we may not have the most up to date code, which is why this is an important topic.
Check it out @ git-scm.com
